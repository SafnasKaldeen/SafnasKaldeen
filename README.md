### 💡 A project I'm proud of

**Sinhala Text-to-Speech (Final Year Project)** — Sinhala is spoken by 16M+ people but had no robust TTS system due to severe dataset scarcity. I trained and compared three architectures (Tacotron 2, VITS, WaveNet) to find which handled low-resource conditions best, validated results with MOS listening tests and speaker-adaptation stress tests, and open-sourced the dataset for other low-resource-language researchers.
→ `PyTorch` `Coqui TTS` `VITS` `Tacotron2` `WaveNet` `TensorBoard`
